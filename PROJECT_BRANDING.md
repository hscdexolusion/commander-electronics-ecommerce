
# 🎨 COMMANDER ELECTRONICS BRAND SYSTEM

Based on:

* Electrical tools/equipment niche
* Deep red + black preference
* Modern ecommerce look
* Industrial/professional feel

---

# 1️⃣ COLOR SYSTEM

## Primary Brand Color

```css
#B91C1C
```

Tailwind:

```tsx
red-700
```

Use for:

* Primary buttons
* CTA sections
* Active links
* Sale badges
* Cart badge

# 2️⃣ TYPOGRAPHY SYSTEM

Let's define it now so we don't forget.

---

## Hero Title

```tsx
text-5xl md:text-6xl font-bold
```

Example:

```text
Power Your Work With
Professional Electrical Tools
```

---

## Section Titles

```tsx
text-3xl font-bold
```

Examples:

```text
Featured Products
Shop by Category
```

---

## Card Titles

```tsx
text-lg font-semibold
```

---

## Body Text

```tsx
text-zinc-400
```
